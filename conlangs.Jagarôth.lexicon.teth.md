teth :: 3s personal pronoun; 3rd person sapient pronoun :: he/she/it
tethen :: 3ps personal pronoun; they
sapient == capable of speech

verbal subject and nominal possessor suffixes are:

-eth :: 3rd singular sapient verbal ending
-ethen :: 3rd plural sapient ""     ""

-eth :: 3ss possessor ending :: his/her/its
-ethen :: 3ps possessor :: their

-ethen :: 3ss possessor plural == -ethen :: 3p possessor
kenetethen :: his dogs or their dog
